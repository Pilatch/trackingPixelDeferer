trackingPixelDeferrer

An object that queues up tracking pixels to be fired in a way that does affect page performance timings.

You really should only ever need to use its .add method

Example

Your page needs to fire a pixel to tracker.com 

trackingPixelDeferrer.add("http://tracker.com/trackme?userdata=1234567890")

Methods

.add(urlOrCallback, htmlTagType)

Add a pixel to the queue to be automatically rendered to the page later.

Parameters:
Name Type Argument Default Description
urlOrCallback function | String

Either a URL to the tracking pixel, or a callback to be fired.
htmlTagType String <optional>
 "img"

The type of html tag that will be rendered. Possible values include: "script", "iframe", "img".

.fire()

Automatically called after the window loaded event. This allows you to fire the pixels manually if the page is already completely loaded.
